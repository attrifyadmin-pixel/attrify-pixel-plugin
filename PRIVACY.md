# Privacy Policy for Attrify Pixel Plugin                                                                                                      
                                                                     
  Last Updated: March 26, 2026

  ## Data Collection

  This extension does not collect, store, or transmit any user data.                                                                             
   
  ## How the Extension Works                                                                                                                     
                                                                     
  - Monitors Attrify Pixel SDK events locally within your browser
  - All event data is displayed only in the Chrome DevTools panel
  - No data is sent to external servers
  - No data is stored persistently (all data is cleared when DevTools is closed)
  - No analytics, tracking, or telemetry of any kind                                                                                             
   
  ## Permissions                                                                                                                                 
                                                                     
  The extension requires access to all URLs (`<all_urls>`) to inject a monitoring                                                                
  hook on websites where you are testing the Attrify Pixel SDK. This permission
  is used solely for local debugging purposes.                                                                                                   
                                                                     
  ## Third-Party Services

  This extension does not use any third-party services, analytics, or tracking tools.

  ## Contact

  For questions about this privacy policy, please contact: support@attrify.com                                                                   
   
  ## Changes to This Policy                                                                                                                      
                                                                     
  We will update this policy if our data practices change.
  The "Last Updated" date will reflect any modifications.
